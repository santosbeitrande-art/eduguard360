# EduMarket MZ - Plataforma de Educação e Serviços Profissionais

## 📱 O que é?

Uma plataforma de **marketplace online** que conecta educadores, consultores e profissionais moçambicanos com estudantes e clientes, permitindo:

- ✅ **Criar e vender cursos online**
- ✅ **Oferecer serviços profissionais** (consultoria, design, programação, etc.)
- ✅ **Ganhar renda recorrente** (passiva e ativa)
- ✅ **Trabalhar remotamente de qualquer lugar**
- ✅ **Acessível via mobile** (a maioria dos moçambicanos usa apenas mobile)

---

## 💰 Como Gera Renda?

### Para Educadores/Criadores:
1. **Criar um curso** (teoricamente, prática, vídeos)
2. **Definir preço** (ex: 199 MT por aluno)
3. **Ganhar comissão** (80-90% do valor de cada venda após plataforma receber 10-20%)
4. **Ganho ilimitado** - quanto mais alunos, mais renda

### Para Profissionais de Serviços:
1. **Oferecer serviço** (consultoria, design, desenvolvimento, etc.)
2. **Definir taxa horária** (ex: 50 MT/hora)
3. **Receber pagamentos** diretos ou por projeto
4. **Construir reputação** com ratings e reviews

### Exemplo de Renda (Educador):
- Curso custa 299 MT
- Ganha 239 MT por aluno (80% após comissão)
- 100 alunos = **23,900 MT** em renda passiva
- 500 alunos = **119,500 MT** 
- 1000 alunos = **239,000 MT**

## 🚀 Funcionalidades Principais

### 1. **Marketplace de Cursos**
- Catálogo com filtros por categoria
- Sistema de avaliação (ratings/reviews)
- Preview de conteúdo
- Pagamento seguro

### 2. **Marketplace de Serviços**
- Perfis de profissionais
- Portfolio/exemplos de trabalho
- Sistema de matching cliente-profissional
- Histórico de transações

### 3. **Dashboard do Vendedor**
- Estatísticas de ganhos
- Gestão de cursos/serviços
- Análise de estudantes/clientes
- Relatórios de performance

### 4. **Sistema de Pagamentos**
- M-Pesa (popular em Moçambique)
- Transferência bancária
- Cartão de crédito (via Stripe/PayPal)
- Saque mínimo de 1000 MT

### 5. **Certificados Digitais**
- Certificados de conclusão
- Validação via blockchain (opcional)
- Compartilhável em LinkedIn

## 📊 Modelo de Negócio

### Receita da Plataforma:
- **Comissão por curso**: 15-20% de cada venda
- **Comissão por serviço**: 10-15%
- **Publicidade**: Marcas podem anunciar cursos/serviços
- **Plano Premium**: Educadores pagam taxa mensal para featured placement
- **Certificados Premium**: 50 MT por certificado com branding custom

### Exemplo de Receita:
- 1000 cursos na plataforma
- 100 vendas/mês por curso em média
- Preço médio: 250 MT
- 100,000 vendas/mês × 250 MT × 15% = **3,750,000 MT/mês**

## 🔧 Tecnologia Stack

### Frontend:
- React + TypeScript
- Tailwind CSS (responsivo, mobile-first)
- Lucide Icons
- Redux/Zustand para estado

### Backend:
- Node.js + Express
- PostgreSQL/MongoDB
- Stripe/M-Pesa API
- AWS S3 (para vídeos/conteúdo)

### Infraestrutura:
- Vercel (frontend) ou AWS
- Digital Ocean (backend)
- CDN para vídeos (Cloudflare)

## 🎬 Como Começar

### 1. **Para Educadores:**
```
1. Registar-se na plataforma
2. Criar um curso (título, descrição, conteúdo)
3. Fazer upload de vídeos/materiais
4. Definir preço e horários
5. Publicar e começar a ganhar
```

### 2. **Para Profissionais:**
```
1. Criar perfil detalhado
2. Fazer upload de portfolio
3. Definir taxa/preços
4. Receber propostas de clientes
5. Completar trabalhos e ganhar
```

### 3. **Para Clientes:**
```
1. Procurar curso/serviço
2. Ver avaliações e preview
3. Efetuar pagamento (seguro)
4. Acessar conteúdo/serviço
5. Deixar avaliação/feedback
```
