## 🎯 User Journey Visual

```
ENTRADA
   │
   ▼
Landing Page
   │
   ├─→ Hover "Portais" (desktop)
   │   └─→ Vê dropdown com 4 portais
   │
   ├─→ Clica "Explorar Portais" (hero)
   │   └─→ Vai para /portais
   │
   ├─→ Vê PortalsPromoSection
   │   └─→ 2 cards grandes (Seg. + Edu.)
   │
   └─→ Clica "Ver Todos os Portais"
       └─→ /portais (Portal Hub)

Portal Hub (/portais)
   │
   ├─→ Vê 4 portais em grid
   │   ├─ 2 com botão "Acessar"
   │   └─ 2 com botão "Em Breve"
   │
   ├─→ Clica "Acessar Portal"
   │   └─→ Entra no portal desejado
   │
   └─→ Clica "Em Breve"
       └─→ Modal de notificação
           └─→ Preenche email
               └─→ Recebe notificação

Portal de Destino
   └─→ /sistema (Segurança)
       ou /edumarket (EduMarket)
```
